# sistem_inventory_pelindo
Sistem Informasi Manajemen Inventaris dan Aset yang dikembangkan dalam program magang di PT Pelindo Multi Terminal, menggunakan React sebagai frontend, Lumen (REST API) sebagai backend, dan MySQL (XAMPP) sebagai database untuk mendukung pendataan, peminjaman, pengembalian, pemeliharaan, serta pelacakan riwayat aset secara terpusat dan terstruktur.
